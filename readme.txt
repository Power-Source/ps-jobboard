Verbinde Menschen, Projekte und Fachexperten – eine professionelle und flexible WordPress Job-Börse.

== Description ==

**PS Job-Board** ist die ultimative Lösung, um eine professionelle Job-Börse in WordPress zu betreiben. Egal ob Du Freelancer, Experten oder Professionelle mit Projekten und Jobs verbinden möchtest – PS Job-Board bietet Dir alle Tools, die Du brauchst.

### 🎯 Hauptfunktionen

**Veröffentliche einen Job**
Veröffentliche Projekte über eine intuitive, reaktionsschnelle Front-End-Oberfläche. Beschreibe Dein Projekt, lege ein Budget fest, setze eine Deadline und hänge Spezifikationsdateien an. Erstelle eine Webseite, auf der qualifizierte Fachleute gerne mit Dir an Deinem nächsten großen Projekt arbeiten.

**Präsentiere Deine Fähigkeiten**
Alles, was Du brauchst, um Deine Fähigkeiten zu zeigen. Teile einen Avatar, eine Biografie, Kontaktinformationen, Beispielarbeiten und Links zu sozialen Profilen. Wähle manuell die perfekte Person basierend auf Erfahrung, Portfolio und Empfehlungen anderer.

**Professionelle Verwaltung**
Verwalte Jobs, Experten und alle damit verbundenen Daten einfach über ein intuitives Admin-Panel. Vollständige Kontrolle über Einstellungen, Kategorien und Custom Fields.

**Erweitere die Funktionalität**
Aktiviere zusätzliche Erweiterungen, um PS Jobboard an Deine Bedürfnisse anzupassen. Integriere MarketPress zur Monetarisierung mit Guthabenpaketen und vieles mehr.

== Installation ==

1. Lade das Plugin hoch oder installiere es über das WordPress Plugin-Verzeichnis
2. Aktiviere das Plugin im Bereich "Plugins" Deines WordPress Dashboards
3. Gehe zu **Jobs > Getting Started** um die ersten Schritte zu unternehmen
4. Passe die Einstellungen nach Deinen Anforderungen an
5. Erstelle Jobs und Experten und starte mit Deinen Nutzern!

== Verwendung ==

Nach der Installation siehst Du zwei neue Menüpunkte in Deinem WordPress Dashboard:
- **Jobs** – Verwaltung aller Job-Einträge
- **Experten** – Verwaltung aller Experten-Profile

Jeder Bereich hat sein eigenes Einstellungsmenü:
- Allgemeine Einstellungen
- Job-Einstellungen
- Experten-Einstellungen

Schau in unserem detaillierten [Handbuch](https://github.com/Power-Source/ps-jobboard) für weitere Informationen.

== Erweiterungen ==

PS Job-Board unterstützt folgende Erweiterungen:

- **Advanced Search** – Erweiterte Suchfunktion für Job- und Experten-Listings
- **Advanced Textarea** – WYSIWYG-Editor für Detail-Felder
- **Custom Layout** – Dynamisches Layout für Listings
- **Expert Demo Data** – Test-Datensätze für Experten
- **Job Demo Data** – Test-Datensätze für Jobs
- **Pages Manager** – Verwaltung virtueller Seiten
- **Custom Fields** – Custom Fields für Anpassungen
- **MarketPress Integration** – Monetarisierung via Guthabenpakete

== Häufig gestellte Fragen ==

= Kann ich die Felder anpassen? =
Ja! Mit den Custom Fields Erweiterungen kannst Du zusätzliche Felder zu Jobs und Experten hinzufügen.

= Ist das Plugin mehrsprachig? =
Ja, PS Job-Board ist vollständig übersetzbar und unterstützt mehrere Sprachen über WordPress-Standard-Tools.

= Kann ich damit Geld verdienen? =
Ja! Mit der MarketPress Integration kannst Du Guthabenpakete anbieten und das Plugin monetarisieren.

= Kann ich das Design anpassen? =
Selbstverständlich! PS Job-Board ist vollständig an Dein Theme anpassbar. Passe Farben, Icons und Layouts an Deine Marke an.
